# Cyber-Crime-Reporting-Awareness-Portal
A web application for reporting cyber crimes, tracking complaints, and spreading awareness through alerts and data visualization.

## Features
- File cyber crime complaints
- Track complaint status
- View scam alerts
- Safety tips for users
- Data visualization dashboard

## Tech Stack
- HTML, CSS, JavaScript
- React JS
- JSON

## How to Run
1. Clone the repo
2. Run `npm install`
3. Run `npm start`
